## Deploy Steps (Copy/Paste)

### 1️⃣ Go to Hugging Face
```
https://huggingface.co/spaces
```

Click: **"Create" → "New Space"**

### 2️⃣ Fill Form
```
Space name:   alhaq-chatbot
License:      OpenRAIL-M
SDK:          Docker
Visibility:   Public
```

Click: **"Create Space"**

### 3️⃣ Upload 3 Files
Click: **"Files" → "Add file" → "Upload files"**

Upload from `d:\Projects\alhaq-website\hf-space-chatbot\`:
- [ ] `app.py`
- [ ] `requirements.txt`
- [ ] `Dockerfile`

### 4️⃣ Wait for Build
Status should show "Running" (2-5 min)

### 5️⃣ Get Embed Code
Click: ⋯ (three dots) → **"Embed this Space"**

Copy the iframe code

### 6️⃣ Add to Website
Paste into any page (e.g., `help.html`):

```html
<iframe
    src="https://alhaq-hf-alhaq-website-chatbot.hf.space"
    frameborder="0"
    width="100%"
    height="600"
    style="border: 1px solid #ddd; border-radius: 8px;"
></iframe>
```
